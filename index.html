Операційна система Linux

Коротка інформація про ОС GNU/Linux

Linux — це вільна операційна система з відкритим кодом, розроблена Лінусом Торвальдсом у 1991 році. Вона базується на Unix і використовується на серверах, робочих станціях, мобільних пристроях та вбудованих системах.

Найпопулярніші дистрибутиви Linux

  • Ubuntu
  • Debian
  • Fedora
  • CentOS
  • openSUSE
  • Arch Linux

Приклади команд роботи в Linux CLI

  • pwd — показати поточний каталог
  • echo — вивести текст або змінну
  • ls — показати список файлів
  • cd — змінити каталог
  • touch — створити порожній файл
  • mkdir — створити каталог
  • cp — копіювати файли і каталоги
  • mv — переміщувати або перейменовувати файли
  • rm — видаляти файли і каталоги
  • history — показати історію команд

BASH-скрипт, що імітує авторизацію в системі

#!/bin/bash
echo "Введіть логін:"
read user
echo "Введіть пароль:"
read -s pass
if [[ "$user" == "admin" && "$pass" == "linux123" ]]; then
  echo "Авторизація успішна! Ласкаво просимо, $user."
else
  echo "Невірний логін або пароль."
fi

Цей простий скрипт демонструє базову перевірку логіна і пароля у текстовому режимі.
