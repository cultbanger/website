Операційна система Linux

Коротка інформація про ОС GNU/Linux

Linux — це вільна операційна система з відкритим кодом, створена Лінусом Торвальдсом у 1991 році. Вона базується на архітектурі Unix і широко використовується на серверах, комп’ютерах, мобільних пристроях і вбудованих системах.

Найпопулярніші дистрибутиви Linux

  • Ubuntu
  • Debian
  • Fedora
  • CentOS
  • openSUSE
  • Arch Linux

Приклади команд роботи в Linux CLI

  • pwd — показати поточний каталог
  • echo — вивести текст
  • ls — список файлів
  • cd — змінити каталог
  • touch — створити файл
  • mkdir — створити каталог
  • cp — копіювання
  • mv — переміщення/перейменування
  • rm — видалення
  • history — історія команд

BASH-скрипт, що імітує авторизацію

#!/bin/bash
echo "Введіть логін:"
read user
echo "Введіть пароль:"
read -s pass
if [[ "$user" == "admin" && "$pass" == "linux123" ]]; then
  echo "Авторизація успішна! Вітаємо, $user."
else
  echo "Невірний логін або пароль."
fi
